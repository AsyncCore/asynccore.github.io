Bucle While

Vegeta777 Java
Los bucles do-while y while en Java te permiten, como su nombre indica (while significa mientras), repetir una acción en un bucle siempre y cuando se cumpla una condición booleana de control. Es posible que el código contenido en un bucle while no se ejecute, porque no se cumpla la condición. No obstante, el código contenido en un bucle do-while se ejecuta, por lo menos, una vez.
